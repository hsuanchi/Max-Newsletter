# [Max 行銷誌電子報](https://article.maxlist.xyz/)

### 1. Side project 原由：

很喜歡的一句話是：「我們都是環境下的產物」，說明其實看似每個決定都是自己在做選擇，但其實促成每個決定的背後都是過去所獲得的資訊累積。
這專案希望能讓自己更留意平時被餵食的資訊，所以精選了 21 個平常會關注的網站，用爬蟲追蹤資訊並整理電子報，於每週四下午四點，帶大家掌握行銷大小事。


### 2. 使用技術：

Crawler:
* Beautifulsoup
* Feedparser

Mail Server:
* Smtplib

Web Server:
* Nginx
* uWSGI

Testing:

* Mock
* Coverage

FrontEnd:
